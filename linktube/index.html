LinkTube LinkTube+ LinkTube++


LinkTube is a JavaScript application that replaces the videos embedded in web pages with a link to the video page. There are two extra versions that show the video title and the streams for YouTube videos.



Installation
Read the following information on how to install the script in different browsers.


To work in Firefox based browsers the script has to be installed with one of Greasemonkey, Tampermonkey or Violentmonkey add-ons from Mozilla Add-ons. Or install the LinkTube add-on from Mozilla Add-ons or manually download and install it from here.

To work in Chromium based browsers the script has to be downloaded and dragged and dropped in the extensions page or installed with the Tampermonkey or Violentmonkey extension from Chrome Web Store. Or install the LinkTube extension from Chrome Web Store or manually download and install it from here.

To work in Pale Moon or Basilisk it is required to install the Greasemonkey extension.

To work in Falkon (QupZilla) it is required to enable the GreaseMonkey extension.

To work in Opera the script has to be downloaded and dragged and dropped in the extensions page or installed with the Tampermonkey or Violentmonkey extension from Opera Add-ons. Or manually download and install the LinkTube extension from here. or versions older than 15 download and save the script in the JavaScript files directory and enable "User JavaScript on HTTPS" in "opera:config" or use the Violentmonkey extension.

To work in Safari it is required to install the Tampermonkey extension.
